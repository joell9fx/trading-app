# VisionEdge FX v1.1 — Feature & System Planning Outline

30-day roadmap to kick off after v1.0 is stable for 2+ weeks.

## Goals
- Expand utility for active traders; raise automation and analytics depth.
- Improve retention via community, mentorship, and onboarding polish.
- Strengthen reliability, observability, and compliance posture.

## Platform Enhancements
- Community Hub: threaded replies + emoji reactions to clarify context and increase engagement.
- Mentorship: booking calendar, mentor tiers, Stripe recurring subscriptions.
- Signals: analytics dashboard (hit rate, ROI, risk metrics) auto-populated from signal posts.
- Funding Accounts: progress tracker with prop-firm API sync; optional webhooks (MyForexFunds, TFT, etc.).
- Courses: video player with Supabase storage; per-module quiz/score + progress save.

## AI & Automation Upgrades
- AI Trade Analyzer (🔥 high): users upload screenshots → model identifies liquidity zones & bias; target <5s response.
- Auto-Summary Bot (medium): weekly digest in community hub (“Top trades”, “Most liked posts”).
- Mentor AI Assistant (medium): custom GPT inside mentorship chat for FAQ/scheduling help.
- Admin AI Audit (low): flag repetitive or low-quality signals automatically.

## Data & Analytics
- Usage: DAU/WAU, retention, conversion to paid unlocks.
- Financial: revenue by product, refund ratio, subscription churn.
- Community: message volume, most active channels, response time.
- System Health: latency, error rate, uptime trend dashboards.

## Technical Upgrades
- Caching layer: Redis for API speed and rate-limit handling.
- Message queue: Supabase Functions or RabbitMQ for async webhooks + AI jobs.
- Image/asset CDN: move static files to Cloudflare R2 or Supabase Storage.
- Performance audit: Lighthouse ≥95 desktop, ≥85 mobile; remediate blockers.
- Testing: Playwright E2E covering auth, chat, checkout, verification.

## UX & Brand
- Unified dashboard layout (mobile-first grid); refine typography/spacing in black/gold theme.
- Onboarding walkthrough for new sign-ups.
- Color accessibility to WCAG AA.
- Short explainer videos for each core module.

## Compliance & Reliability
- Review GDPR/UK data retention and cookie policy pages.
- Add 2FA toggle in user settings (Supabase Auth OTP).
- Schedule weekly backup verification test.
- Error-budget policy: <1% unhandled failures/week.

## Suggested Sprint Timeline
- Week 1–2: threads + reactions, signal analytics, mentor calendar/tiers/subscriptions.
- Week 3–4: AI Trade Analyzer, UX refresh, caching layer + queue/CDN setup.
- Week 5: testing, docs, deployment pipeline polish.
- Week 6: beta rollout, feedback, tag v1.1 RC.

## Deliverables for v1.1 Tag
- Community and mentorship upgrades live; AI Analyzer stable (<5s avg response).
- Analytics dashboards (usage, financial, community, system health) in place.
- 95% of validation checks automated in CI; Playwright suite green.
- Docs updated: `/docs/features-v1.1.md`, `/docs/changelog.md`.
- Release tag: `v1.1-launch`.
