# Trading Academy - Preview Site

## 🚀 Quick Start

### Option 1: Next.js Preview (Recommended)
```bash
# Start the development server
npm run dev

# Visit the preview page
http://localhost:3000/preview
```

### Option 2: Static HTML Preview
```bash
# Open the static HTML file directly in your browser
open preview-site.html
```

## 📋 Preview Features

The preview site showcases:

- **Platform Overview**: Complete feature breakdown
- **Quick Navigation**: Direct links to all app sections
- **Statistics**: Key metrics and achievements
- **Technology Stack**: Modern development tools used
- **Responsive Design**: Works on all devices

- **Main Site**: `http://localhost:3000/`

- **Mentoring**: `http://localhost:3000/mentoring`
- **Pricing**: `http://localhost:3000/pricing`
